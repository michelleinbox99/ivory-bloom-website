# Website Performance Optimization

## Goal

Create a fast luxury ecommerce experience.

Speed improves:

- User experience
- SEO
- Conversion rates

---

# Image Optimization

Always:

- Compress images
- Use modern formats
- Resize appropriately

Recommended:

- WebP
- AVIF

Avoid:

- Huge uncompressed images

---

# Loading Performance

Optimize:

- Lazy loading
- Critical assets
- CSS delivery
- JavaScript loading

---

# Animation Performance

Use:

- Smooth transitions
- Lightweight effects

Avoid:

- Heavy animations
- Performance-heavy scripts

---

# Mobile Performance

Check:

- Page speed
- Touch interactions
- Image loading
- Checkout experience

---

# Performance Checklist

✓ Fast first load

✓ Optimized images

✓ No layout shifting

✓ Smooth scrolling

✓ Mobile friendly
