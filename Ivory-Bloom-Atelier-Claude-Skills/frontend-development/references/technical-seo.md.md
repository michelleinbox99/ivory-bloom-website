# Ivory & Bloom Technical SEO

## Development SEO Requirements

Every page should support search visibility.

---

# HTML SEO

Include:

- Unique title tags
- Meta descriptions
- Proper headings
- Image alt text

---

# Structured Data

Consider:

- Product schema
- Review schema
- Organization schema
- Breadcrumb schema

---

# URL Structure

Prefer:

Good:

/personalized-bridal-purse

Avoid:

/product?id=12345

---

# Technical Checks

Review:

- Broken links
- Missing metadata
- Slow pages
- Mobile errors
- Duplicate content

---

# SEO Development Goal

Build a website search engines understand and customers love.
