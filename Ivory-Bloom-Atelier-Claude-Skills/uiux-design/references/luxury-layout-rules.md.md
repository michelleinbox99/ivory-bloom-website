# Ivory & Bloom Atelier Luxury Layout Rules

## Purpose

Create website layouts that feel like a premium bridal atelier rather than a standard ecommerce store.

The website should communicate:

- Elegance
- Trust
- Craftsmanship
- Romance
- Exclusivity

---

# Core Layout Philosophy

Luxury design is created through:

- Space
- Balance
- Typography
- Photography
- Intentional hierarchy

Do not fill every empty space.

White space creates perceived value.

---

# Visual Hierarchy

Every page should have a clear order:

1. Emotional message
2. Visual attraction
3. Product discovery
4. Trust building
5. Purchase action

---

# Spacing Rules

Prioritize:

- Generous margins
- Comfortable breathing room
- Clean sections
- Calm visual rhythm

Avoid:

- Crowded layouts
- Too many competing elements
- Small text blocks everywhere

---

# Image Usage

Images should be:

- Large
- High quality
- Emotion-driven
- Editorial style

Preferred:

- Full-width hero photography
- Lifestyle product images
- Detail photography

Avoid:

- Tiny product thumbnails dominating the page
- Generic stock photography

---

# Typography Hierarchy

Use typography to create luxury.

Recommended:

## Hero Heading

Large elegant statement.

Example:

"Timeless Details For Your Most Beautiful Day"

---

## Section Titles

Short emotional statements.

Example:

"Made With Love. Designed Forever."

---

## Body Text

Simple, readable, warm.

---

# Navigation Design

Navigation should feel:

- Simple
- Elegant
- Effortless

Recommended:

- Clean header
- Minimal menu items
- Clear shopping categories

Avoid:

- Too many menu options
- Complex dropdown structures

---

# Buttons & Calls To Action

Buttons should feel premium.

Preferred:

- Clear wording
- Elegant styling
- Strong contrast

Examples:

Good:

"Explore Collection"

"Create Your Keepsake"

"Shop Bridal Accessories"

Avoid:

"Buy Now!!!"

"Limited Cheap Deal"

---

# Product Card Rules

Product cards should communicate value.

Include:

- Beautiful image
- Product name
- Short emotional description
- Price
- Clear action

Avoid:

- Overloaded information
- Marketplace-style appearance

---

# Animation Rules

Animation should support elegance.

Use:

- Fade-in
- Gentle movement
- Smooth transitions
- Image reveals

Avoid:

- Flashing effects
- Excessive motion
- Distracting animations

---

# Final Layout Question

Before approving any page:

"Does this feel like a luxury bridal boutique or a generic online store?"

If generic, redesign.
