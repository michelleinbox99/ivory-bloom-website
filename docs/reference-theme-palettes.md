# Reference: Retired Atelier Customizer Palettes

The customer-facing Theme Customizer (formerly `js/customizer.js` + `css/07-customizer.css`)
was retired in Website 2.0 Phase B — a luxury brand's palette is a fixed trust signal,
and letting visitors repaint the site undercut it (audit finding #15; brand-identity
guidelines: "protect brand consistency").

The four palettes are archived here as reference — useful if a seasonal campaign
micro-site or a wedding "mood palette" preview feature is ever wanted. The first
palette (Blush Pearl & Cocoa) remains the live site palette in `css/00-tokens.css`.
